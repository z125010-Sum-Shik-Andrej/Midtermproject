Research in AI

Neural Network from Scratch for House Price Prediction

In this University project we wanted to create an AI that can predict the pricing of Real Estate, for this we had different models and evaluated how well they perform for a given (self-obtained) dataset, we compared a linear, polynomial and neural network, which we build from Scratch.

For more information on the results and more insight into our neural network as well as features of the famous library TensorFlow, please read the detailed report here: PricePreReportFinal.pdf

Finetuning GPT-2 Model for Creating First Good Messages

In this University project we created a dataset of given user profiles alongside good first messages for these profiles, we then took a pretained version of gpt-2 and finetuned it with our dataset.

For more information on the results and on the project itself please go to the projects section or please read our detailed documentation available at: IceBreakerReportChanged.pdf
